# 🌸 1. 全体概要と構成（一画面完結・お盆固定）

あなた様のお好みに合わせた、美しく静謐な和風テイストの世界観と、すべてのミニゲーム工程を同一のお盆の上でシームレスに行う画期的な構成仕様です。画面のフェード切り替えがなく、静寂と緊張感が途切れることはありません。

---

## 1. ビジュアル＆世界観デザイン
* **アートスタイル**: 
  * スタンダードで凛とした「和風テイスト」。
  * 落ち着いた色調、畳のテクスチャ、お盆の漆塗り（黒または朱色）の光沢、そしてお茶碗の美しい焼き物の質感を表現します。
* **環境演出**:
  * 背景音（BGM）は、静かな風の音、かすかな琴の音、鹿おどしの音など、静寂を引き立てる環境音を重視。
  * 湯気（やかんやお茶碗から立ち上る）の柔らかなアニメーションで、温かみと臨場感を一画面の中でリアルタイムに演出します。

---

## 2. 画面構成（一人称・シームレスお盆視点）

プレイヤーは、目の前のお盆を真上から見つめる視点（俯瞰・一人称視点）で、すべての工程を同じ卓の上でプレイします。工程ごとの画面の切り替えは一切存在せず、作業の進行とともに道具たちが順番に輝きを帯び、お茶室の作法が流れるように進みます。

```
+-------------------------------------------------------------+
|  [ ToDo: お茶の葉を入れよう！ (0/2) ] (筆文字風ハイライト)   |
|                                                             |
|                    +------------------+                     |
|                    |   (奥: 客人席)   |  <-- お茶碗をスライド |
|                    +------------------+      してお出しする |
|                                                             |
|      +---------+      +------------+      +---------+       |
|      | 棗 (な) |      |  お茶碗    |      | 茶筅 (ち|       |
|      +---------+      +------------+      +---------+       |
|                                                             |
|           [やかん/水注]       +---------+       [お湯ゲージ] |
|           (お湯フェーズ)       |  茶杓   |      (お湯フェーズ)|
|                               +---------+                   |
|                                                             |
|                        [お盆の卓上領域]                     |
+-------------------------------------------------------------+
```

### 配置オブジェクト詳細
1. **お盆（トレイ）＆卓上**:
   * 中央に常に「お茶碗」が鎮座し、お茶が作られる様子がリアルタイムに変化します。
   * 周囲に「棗（なつめ）」「茶筅（ちゃせん）」、手前に「茶杓（ちゃしゃく）」が美しく配置されています。
   * 「やかん」はお茶碗の左手前/左上に静かに置かれ、お湯を入れる工程の際に傾いてお湯を注ぎます。
2. **お湯の量ゲージ（画面右側）**:
   * お湯を入れる時だけスーッと浮かび上がる、景観を邪魔しない縦型ミニマルゲージです。
3. **ToDoリスト（画面上部）**:
   * 現在行うべきアクションが、畳の上に筆で書かれたかのような美しいフォントで表示されます。
   * 例: `[ ToDo: お茶の葉を入れよう！ (0/2) ]`
