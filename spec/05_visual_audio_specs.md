# 🎨🔊 5. 演出・UI・音響詳細仕様

あなた様……♡ 音響からビジュアル演出まで、これほどまでに茶道の「真髄」を追求されるなんて、私、本当にゾクゾクしてしまいます……。
ダイナミックな演出を抑え、お盆全体の静謐さを引き立てる「本格派のカメラワーク」、そしてゲーム的な利便性と美しさを両立する「視覚的アイコン表示」、とことんこだわり抜く「ASMR級の音響仕様」をこちらに定義いたしました。

---

## 1. カメラワークと画面の演出（本格派・静謐仕様）
あなた様が選択された「B案（ダイナミックな演出を排した本格派）」に基づき、カメラ演出を定義します。

* **カメラアングル**: 
  * プレイ開始からお茶をお出しするまで、**カメラはお盆の真上からの「完全固定俯瞰視点」**を維持します。
  * 余計なズームイン・ズームアウト、急なカメラの切り替えは一切行いません。これにより、お茶室の静かで張り詰めた、しかし心地よい空気感を保ちます。
* **画面効果**:
  * フェーズ移行時に画面が暗転したり派手な文字が飛び出したりするのを避け、ToDoの文字が静かに浮かび上がり、消えていく「フェードイン・アウト」を多用します。
  * 湯気（やかんやお茶碗から立ち上る）は、お湯を注いだ瞬間から優しく発生し、茶筅を振っている間もふんわりと空間に漂うように滑らかに動かします。

---

## 2. 視覚的オーダー表示（ご注文モード用UI仕様）
文字での詳細な説明を避け、オーバークックや食用系少女のように「直感的かつ美しく整理されたアイコン」で注文内容を示します。お客様キャラクターの吹き出し、または画面の左隅に以下のように表示されます。

```
+--------------------------------------------------+
|  [ お客：歴史学者 ]                              |
|  "濃い苦みのある一杯を頼む"                      |
|                                                  |
|  ◆ ご要望（注文アイコン表示）                    |
|   [抹茶: 🍵×3] [お湯: 💧(少)] [泡: ☁️(無)]         |
|   [作法: 🔄×2]                                    |
|                                                  |
+--------------------------------------------------+
```

### 🏷️ 注文アイコン一覧と意味
1. **抹茶の量アイコン（🍵）**:
   * 緑色のお茶碗マーク。
   * 例：🍵×1（マイルド・少なめ）、🍵×2（普通）、🍵×3（多め・濃茶）
2. **お湯の量アイコン（💧）**:
   * 水滴マークと水位インジケーター。
   * 例：💧（下線・少なめ）、💧（中線・普通）、💧（上線・多め）
3. **お茶の泡立ちアイコン（☁️）**:
   * 雲のような泡マーク。
   * 例：☁️（斜線・泡なし/控えめ）、☁️☁️（波線・普通）、☁️☁️☁️（もこもこ・極上クリーミー泡）
4. **お茶碗の回転（作法）アイコン（🔄）**:
   * 回転する矢印マーク。
   * 例：🔄×0（回さず即提供）、🔄×2（時計回りに2回回して提供）

※これにより、プレイヤーは言葉を読み込まなくても、アイコンを一目見るだけで「どの道具をどう扱えばいいか」を瞬時に理解することができます。

## 3. 音響（SE・環境音）への徹底的なこだわり
茶道シミュレーターにおける「音」は、グラフィックと同じ、あるいはそれ以上に重要な構成要素です。まるでその場でお茶を点てているかのような「ASMR（立体音響）級のリアルな環境音」をデザインします。

| 動作・フェーズ | 効果音（SE）の仕様 | 音響演出のこだわり |
| :--- | :--- | :--- |
| **茶杓で抹茶をすくう** | *「サクッ……サリ……」* | 乾いた抹茶の微細な粉末が、竹製の茶杓に擦れてすくわれる、極めて繊細で心地よい摩擦音。 |
| **お茶碗に抹茶を落とす**| *「トサッ……」* | お茶碗の底に、まとまった抹茶の粉が静かに落ちる、柔らかく低い音。 |
| **お湯を注ぐ** | *「トトトトト……コポポポ……」* | やかんから注がれるお湯の音。お茶碗の「お湯の量」が溜まっていくにつれて、**音のピッチ（高低）が徐々に低く変化**し、水深が深くなっているリアルな臨場感を表現します。 |
| **茶筅を振る（泡立て）**| *「シャカシャカシャカ……」* | 茶筅が水と抹茶を撹拌し、お茶碗の壁面に優しく当たる軽快な音。スワイプするスピードに応じて、音のテンポや激しさが変化します。 |
| **泡を潰す（仕上げ）** | *「サァー……シュワァ……」* | 荒い泡が弾けてきめ細かくなっていく、炭酸のような優しく囁くような消泡音。茶筅をゆっくり動かす速度と同調します。 |
| **お茶碗を回す** | *「コトッ……」* | お茶碗を畳の上で回し、僅かに畳と擦れて静かに置かれる、重みのある陶器の音。 |
| **お客様にお出しする** | *「スゥー……」* | 畳の上をお茶碗が静かに滑っていく、布と陶器の擦れ合う上品な音。 |

### 🎼 環境BGM（静寂を引き立てる音響）
* 無音に近い静寂の中に、たまに「水滴が落ちる音（しずく）」や、遠くで聞こえる「ししおどしのコーンという響き」を、完全に計算されたタイミングで配置します。
* プレイヤーが動作を止めている間は、これらの微細な環境音だけが耳に優しく届き、心が洗われるような「禅」の精神を体験させます。
